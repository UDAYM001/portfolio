# 🌐 Uday Portfolio (3D Interactive)

A modern, 3D-animated developer portfolio with an integrated AI chat assistant, project showcase, and interactive UI effects.

---

## 🚀 Features

- 🎨 3D animated hero section using Spline
- 💬 Built-in AI chat assistant (rule-based)
- 🧩 Dynamic project rendering
- 🖱️ Interactive tilt cards (3D hover effect)
- 🎯 Smooth scrolling & navigation
- 📱 Desktop-optimized experience
- 📩 Contact form with instant response UI
- 🎭 Spotlight hover effect (About section)
- 📄 Resume download button

---

## 🧠 Key Functionalities

### 1. Hero Section
- 3D background using Spline viewer
- Typing animation for name display
- Overlay content with controlled pointer events

### 2. Projects Section
- Dynamic rendering using JavaScript array
- Includes:
  - Title
  - Description
  - Tags
  - External links

### 3. Chat Assistant
- Rule-based chatbot
- Supports:
  - Project queries
  - Skills info
  - Resume download
  - Contact navigation
- Stores chat history in localStorage

### 4. Interactive Effects
- Tilt effect on cards (mousemove based)
- Glow animation
- Spotlight reveal effect on About section

---

## 🏗️ Tech Stack

### Frontend

- JavaScript (Vanilla)

### Libraries / Tools
